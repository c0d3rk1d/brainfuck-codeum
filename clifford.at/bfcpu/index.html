Brainf*ck Projects

  • The Brainf*ck CPU

    A VHDL CPU design for a native brainf*ck machine. I've also built it in hardware running at 48MHz (using a Xilinx Spartan-II FPGA).


     
  • The Brainf*ck Compiler

    A compiler for a c-like high-level language which generates brainf*ck code (a towers of hanoi with semi-graphical terminal output is included as example).


     
  • Towers of Hanoi

    I've done an implementation of the well-known "Towers of Hanoi" problem in Brainf*ck. The program features semi-graphical terminal output (see screenshot) and does solve the problem by itself (i.e. it doesn't simply output an hardcoded solution).
